# Sample Projects

This directory contains sample projects demonstrating `simple-cookie-service` in different environments and frameworks.

## 1. Node.js App

A simple script using the library in a server context.

```sh
cd node-app
npm install
node index.js
```

## 2. React App (Vite)

A minimal React application using `cookies.set` and `cookies.get`.

```sh
cd react-app
npm install
npm run dev
```

## 3. Angular App

A standalone Angular application.

```sh
cd angular-app
npm install
npm start
```

## 4. Vue.js App (Vite)

A minimal Vue 3 application using the Composition API.

```sh
cd vue-app
```
