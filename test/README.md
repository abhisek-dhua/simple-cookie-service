# Sample Projects

This directory contains sample projects demonstrating `simple-cookie-service` in different environments.

## 1. Node.js App

A simple script using the library in a server context.

```sh
cd node-app
npm install
node index.js
```

## 2. React App (Vite)

A minimal React application using `cookies.set` and `cookies.get`.

```sh
cd react-app
npm install
npm run dev
```

## 3. Angular App

A standalone Angular application.

```sh
cd angular-app
npm install
npm start
```
